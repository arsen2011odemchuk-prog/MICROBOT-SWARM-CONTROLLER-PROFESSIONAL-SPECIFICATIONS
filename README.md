# Microbot Swarm Controller

This is a professional, modular controller for microbot swarms.

## Features
- Dual 7–10 inch screens (FPV + Telemetry)
- AI integration (ChatGPT or local model)
- Long-range LoRa communication
- Modular case with hot-swappable battery
- USB/HDMI connectivity for other robots

## Components
See `docs/components.md` for detailed parts list.

## Installation
1. Assemble motherboard, screens, battery in 3D printed case
2. Connect joystick/buttons via USB
3. Attach LoRa antennas
4. Power on and boot OS

## Usage
- GUI for swarm monitoring
- AI assistant for navigation & analysis
- HDMI input/output for external devices
